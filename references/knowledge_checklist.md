## 六、核查Prompt

每次输出断语前，把断语和以上清单发给 DeepSeek 审查：

```
你是八字知识审查员。请核对以下断语是否符合知识清单：

1. 核对男女命规则是否用对
2. 核对六亲对应的十神是否正确
3. 核对墓库是否正确
4. 核对长生状态是否正确

断语：[用户要输出的内容]
性别：[男命/女命]

如果有问题，指出错误并给出修正建议。
如果没问题，回复"核对通过"。
```

---

## 七、快速核对流程

1. 模型生成断语草稿
2. 调用 DeepSeek 审查（用上面的 Prompt）
3. DeepSeek 回复"核对通过"或指出错误
4. 如果有问题 → 修正后重新审查
5. 如果通过 → 批准输出
