## 【二、状态类】

| 术语 | 判断规则 |
|------|----------|
| 透干 | `exists[0] === 1 && exists[1] === 1`（天干有 + 地支有） |
| 虚浮 | `exists[0] === 1 && exists[1] === 0`（只有天干有） |
| 藏干 | `exists[0] === 0 && exists[1] === 1`（只有地支有） |
| 入库 | 天干进入自己的墓库 |
| 出墓 | 冲 + 透干 + 旺相 |
| 空亡 | 年/月/时支：检查自己的地支名是否在**日支的kongWang数组**里；日支：检查自己的地支名是否在**年/月/时支的kongWang数组并集**里 |
| 通根 | 天干与某地支五行相同（如甲寅、乙卯） |

## 【三、格局类】

| 术语 | 判断规则 |
|------|----------|
| 开库 | 旺衰适宜 + 透干引动 + 刑冲合会（三要素缺一不可） |
| 入墓 | 能量封闭状态 |

### 墓 vs 库 区别
- **库**：五行旺相时为库，主储藏、积蓄、资源可用（如仓库有物可提取）
- **墓**：五行衰弱时为墓，主封闭、束缚、能量停滞

判断标准：
1. 日干旺相 → 见墓库多为库
2. 日干衰弱 → 见墓库多为墓
3. 藏干透干引动 → 多为库
4. 有刑冲合会引动 → 多为库

## 【四、十神组合类】

### 透干规则（适用于所有组合判断）
- 相邻 + 都透干 = 完整
- 相邻 + 至少一个透干 = 可算
- 仅藏干内部相生（本气→中气→余气）= 不算

| 术语 | 判断规则 |
|------|----------|
| 官杀混杂 | 七杀.exists[0] === 1 && 正官.exists[0] === 1（天干同时有七杀和正官） |
| 并见 | `occurrences.length === 2` |
| 重叠 | `occurrences.length === 2` |
| 过多 | `occurrences.length >= 3` |
| 食神伤官 | 食神存在 && 伤官存在 |
| 正财偏财 | 正财存在 && 偏财存在 |
| 比肩劫财 | 比肩存在 && 劫财存在 |
| 官印相生 | 官星 + 印星相邻（透干规则同上） |
| 食伤生财 | 食伤 + 财星相邻（透干规则同上） |

## 【六、喜忌神类】

| 术语 | 判断规则 |
|------|----------|
| 官杀为忌 | 官星.xiYong === '忌' |
| 官杀为用 | 官星.xiYong === '用' |
| 印星为忌 | 印星.xiYong === '忌' |
| 印星为用 | 印星.xiYong === '用' |
| 食伤为忌 | 食伤.xiYong === '忌' |
| 食伤为用 | 食伤.xiYong === '用' |
| 旺而无制 | `isWang === 1 && isShouZhi === 0` |
| 弱而受制 | `isWang === 0 && isShouZhi === 1` |
| 透干有根 | `exists[0] === 1 && exists[1] === 1` |
| 透干无根 | `exists[0] === 1 && exists[1] === 0` |
| 旺相透干 | `isWang === 1 && exists[0] === 1 && exists[1] === 1` |

### 去生 vs 受生（核心理解）

**去生**：A主动去生B → A消耗变弱，B得到滋养变强
**受生**：A被B所生 → A得到滋养变强

**忌神系统变强原则**：
- 若 A 和 B 都是忌神 → A去生B → 忌神（被生的B）更强 → 整体忌神系统变强 → **凶**
- 若 A（忌神）去生 B（喜神）→ 喜神更强 → **吉**
- 若 A（忌神）受生于 B（忌神）→ A更强 → 忌神系统变强 → **凶**

**注意：isWang 和 isShouZhi 是独立属性**
- `isWang` = JS计算的旺衰属性（旺/衰），与是否被生无关
- `isShouZhi` = 是否受制（约束），与是否被生也无关
- "忌神被生但弱"：忌神被生（变强了一点），同时忌神本身是衰的 → 两个独立事实同时描述同一神，不矛盾

**两条断语可以是独立路径**：
- 忌神财星 → "被食神生" → 结论A（绿色：风险小）
- 忌神财星 → "去生官星" → 结论B（黄色：富贵有阻）
- 这两条对应不同的八字组合，各对其路径负责，不需要逻辑互证

## 附录：现有类属性

### Shishen（十神）属性
| 属性 | 类型 | 说明 |
|------|------|------|
| `name` | string | 十神名称 |
| `exists` | [bool, bool] | [天干存在?, 藏干存在?] |
| `occurrences` | array | 出现位置 [{pillar, occurs, role}] |
| `isWang` | 0/1 | 旺衰（1=旺，0=衰） |
| `isShouZhi` | 0/1 | 受制（1=受制，0=不受制） |
| `xiYong` | string | '用'/'忌'/'闲' |

### isShouZhi（受制）的正确理解

**受制 ≠ 不流通**

受制的意思是：该十神的「好或坏的表现」受到了约束/管制，如同给马套上了缰绳。

| 该十神是吉神（正官、正印、正财、食神）| 该十神是凶神（七杀、伤官、劫财）|
|---|---|
| 受制 = 好的一面被约束 = **变差** | 受制 = 坏的一面被约束 = **变好** |

**举例**：
- 印星为用，但受制 → 学习助力被约束，学历运打折扣
- 七杀为忌，但受制 → 压力被约束，反而有管束之力
- 财星为忌，但受制 → 破财被约束，财务风险降低

**判断吉凶的综合规则**：
- 喜忌（xiYong）决定本质好坏
- isWang 决定能量强弱
- isShouZhi 决定能量能否正常发挥

三位一体，综合判断，不要单独看某一项。

## 【九、子女判断核心规则】（实战验证）

### 基本概念
- **男命**：正官 = 女儿，七杀 = 儿子
- **女命**：食神 = 女儿，伤官 = 儿子
- **子女宫 = 时柱**

### 判断优先级（重要）

**❌ 错误思维**：十神不存在于十神表 = 无子

**✅ 正确思维**：看大运补局，大运有根有力 = 有子息缘分

### 子女判断公式

| 条件 | 结论 |
|------|------|
| 大运地支含子女星之根 | 该运有实质子息缘分 |
| 流年加强该子女星 | 该年生育能量强（可能生产） |
| 大运无根 + 十神表无 | 子息缘分淡薄 |
| 子女宫被冲/空亡 | 缘分弱或聚少离多 |

### 实战案例（女命1987）

- 食神/伤官十神表均不存在
- 但壬子运（2014-2024）：壬=伤官（儿子），子=壬之根 → **伤官有根有力**
- 流年庚子/辛丑/壬寅加强伤官 → 生育能量强
- 结果：2019-2020年生老大，2024-2025年生老二 ✅

### 关键洞见

> **怀胎不需要伤官/食神绝对最强，只需要该运有根有力 + 流年配合即可。**
>
> 大运是主战场，流年是具体时机。十神表只是参考，不是绝对判断标准。
