用神忌神喜用神
五大核心概念
用神
→ 核心"药方"，治命局失衡的关键五行
喜神
→ 辅助用神的五行，生助用神
忌神
→ 直接克用神的五行，病症根源
仇神
→ 间接助忌神的五行，忌神的帮凶
闲神
→ 中性，可转化为喜神或忌神
用神四大取法
1. 扶抑取用法（最常用）
身旺
→ 取克泄耗为用（官杀克、食伤泄、财星耗）
身弱
→ 取生助为用（印星生、比劫助）
2. 调侯取用法（优先于扶抑）
生于夏月（巳午未）
→ 取水调候
生于冬月（亥子丑）
→ 取火调候
生于四季（辰戌丑未）
→ 取木/金/水调燥湿
3. 通关取用法
金木相战
→ 取水通关
水火相战
→ 取木通关
土水相战
→ 取金通关
4. 病药取用法
土多金埋
→ 病在土旺，药为木/金/水
木多火塞
→ 病在木旺，药为金/火/土
水多木漂
→ 病在水旺，药为土/木/火
