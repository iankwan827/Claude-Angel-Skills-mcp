一、父母断语
来源：理华千山派28式·父母篇
父亲相关
年干比肩 + 月干比肩 + 偏财不显
→ 父亲早逝
年干伤官 + 月干偏印（枭神）
→ 父亲早逝，母亲身体差
年柱纳音被月柱纳音所克 + 岁运再遇同克
→ 父亲必亡
财星入墓又见冲 + 岁运见库
→ 父寿终
天克地冲见偏财
→ 父亡之期提前
母亲相关
年支正财 + 月支正财
→ 母亲早丧或父母早离异
刑冲破印
→ 母走他乡
二重亡神
→ 母先丧
父母综合
时杀带花
→ 父先亡
二、兄弟姐妹断语
来源：理华千山派28式·兄弟姐妹篇
兄弟夭亡
月干是七杀 + 他处再出现七杀
→ 兄弟中有人夭折
同胞流落
比肩入库，然后逢冲
→ 同胞中有人流落在别人家
无兄弟
月上比劫怕透天
→ 无有兄弟，形影单
兄弟有疾
比劫不透不夭折，但有疾或偏瘫
→ 兄弟姐妹有刑疾
兄弟分离
年月刑冲
→ 父东子西把家搬
兄弟反目
比劫重重克财星
→ 兄弟反目成仇
说明：以上断语均来自理华千山派28式。父母与兄弟姐妹的缘分深浅，需结合八字全局通盘判断，不可单凭一条断语定论。
